# E-Bot WhatsApp 🤖

Bot de gestión de turnos médicos usando Twilio + Flask.

## Funciones

- Solicitar turno
- Consultar turnos
- Mensajes
- Panel admin
- Bloqueo de agenda

## Instalación

```bash
git clone https://github.com/tuusuario/ebot-whatsapp-core.git
cd ebot-whatsapp-core
pip install -r requirements.txt
